Paw Rescue — Landing Page
A responsive landing page concept for a stray animal rescue initiative, built as part of the Codveda Technologies Front-End Development Internship (Level 1 — Responsive Landing Page task).
About the project
"Paw Rescue" is a fictional brand created for practice — the goal was to design and build a fully responsive landing page covering a navbar, hero section, feature highlights, and footer, while keeping the visual identity warm and distinctive rather than templated.
Features
Fully responsive layout (mobile, tablet, desktop)
Custom hero section with a signature glowing paw illustration
Feature cards: Feeding Drives, Emergency Rescue, Medical Aid, Adoption Help
Smooth scroll behavior and scroll-triggered animations
Accessible, semantic markup
Tech stack
React + TypeScript
Tailwind CSS
Vite
Note
This is a demo/practice project built for skill development and portfolio purposes — not a registered NGO. The stats shown on the page are illustrative, not real data.
